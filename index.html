The firmware puts a Linux based operating system on your device. It allows you to install additional software.

If you have just bought the device, please choose the first installation image. If you have already installed LEDE/OpenWRT, use the upgrade images.

This firmware is available in the following versions:

  • Stable
    Well tested, reliable and stable.
  • Unstable
    Candidates for the next stable release.
  • Testing
    Untested, build automatically.
